# http 请求封装库

基于Axios 封装的http请求库

## 快速开始

#### 1、项目安装
- 执行 `npm install` 命令

#### 2、开发时调试
- 执行 `npm start` 命令

#### 3、测试
- 执行 `npm run test` 命令

#### 4、项目构建
- 执行 `npm run build` 命令

#### 5、清理 dist 文件夹
- 执行 `npm run clean` 命令即可

#### 6、预编译vendors库
- 执行 `npm run dll` 命令即可
